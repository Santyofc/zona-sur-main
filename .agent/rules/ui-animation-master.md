# Arsenal de UI y Animación Pro Max - Zona Sur Tech

## Referencias de Estética y Componentes
- **Uiverse.io**: Implementar micro-interacciones, botones con efectos glow/glassmorphism y switches táctiles.
- **CodingStella**: Estándar para formularios de Login y Registro (etiquetas flotantes, validación visual, fondos oscuros premium).
- **nextlevelbuilder/ui-ux-pro-max-skill**: Arquitectura de layouts asimétricos y jerarquía tipográfica moderna.

## Principios de Movimiento y Animación (Framer Motion Engine)
La IA debe emular el comportamiento de las siguientes librerías utilizando Framer Motion:

1. **Rellax (Paralaje)**: Crear profundidad en el scroll para elementos de hardware y servicios.
2. **Dynamics.js & Bounce.js**: Aplicar físicas de "Spring" realistas y rebotes elásticos en CTAs y botones de contacto (+506 87623229).
3. **Barba.js**: Transiciones fluidas de entrada/salida entre rutas (AnimatePresence) para evitar saltos bruscos.
4. **Anime.js & MoJS**: Líneas de tiempo complejas para revelación de contenido y efectos de ráfaga (burst) en acciones exitosas.
5. **ScrollReveal**: Aparición secuencial de tarjetas con rotación 3D al entrar en el viewport.
6. **Wait! Animate**: Insertar retardos calculados entre ciclos de animación para naturalidad.
7. **Awesome-Android-UI**: Traducir la fluidez de gestos móviles a componentes web (Shared Element Transitions).

## Reglas de Implementación
- **Identidad**: Siempre cruzar estas animaciones con los colores corporativos de Zona Sur Tech.
- **Performance**: Priorizar CSS nativo o Framer Motion para evitar sobrecarga de JS externo.
- **Mobile First**: Usar Bottom Sheets y menús inspirados en Spirit para navegación móvil.
